# The Walking Dead Website

Projeto simples desenvolvido para praticar conceitos de front-end e Css e testar conhecimentos adquiridos durante os estudos.

> Observação: o projeto não possui conteúdos oficiais da série. Alguns textos utilizados foram gerados apenas para fins de teste e estruturação visual da página utilizando Lorem Ipsum.

O site é inspirado na série "The Walking Dead" e possui:
- Página com resumo da primeira temporada
- Galeria de imagens
- Área de cadastro
- Links para redes sociais
- Botões interativos e elementos visuais

##  Tecnologias utilizadas
- HTML5
- CSS3

##  O que aprendi
- Estruturação de páginas web
- Criação de abas e seções
- Manipulação de elementos visuais
- Criação de botões interativos
- Organização de layout
- Estilização com CSS

##  Como executar
Basta abrir o arquivo index.html no navegador.
