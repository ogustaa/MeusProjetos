#  Sistema de Votação em Python

Projeto simples desenvolvido em Python que simula uma eleição com três candidatos. O programa recebe os nomes dos candidatos, realiza a votação e ao final exibe um relatório com os resultados. 
O projeto foi na verdade um desafio apos a aula de python, onde realizei alguns exercicios, e achei interessante subir aqui, os exercicios estao em outra pasta.

##  Tecnologias utilizadas
- Python

##  O que aprendi
- Entrada de dados
- Estruturas de repetição (while)
- Estruturas condicionais (if / elif / else)
- Contagem de votos
- Cálculo de porcentagem
- Lógica de programação

##  Como executar
Execute o arquivo .py no VS CODE ou em outro lugar usando Python.  
Primeiro informe o nome dos três candidatos e depois realize a votação utilizando o menu exibido no terminal.  
a cada numero que voce digitar, um candidato recebera um voto, então se eu digitar tres vezes o numero 1, o candidato 1 recebera 3 votos, 
assim que voce terminar de estabelecer os votos dos candidatos, digite 0 para encerrar a votação.

`Para encerrar a votação, digite 0.`
