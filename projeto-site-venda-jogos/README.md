# Loja de Jogos Online

Projeto simples para teste, criação de um site para compra de jogos online, desenvolvido para praticar conceitos de front-end e manipulação de elementos com JavaScript.

O projeto possui funcionalidades básicas como:
- Adição de itens ao carrinho
- Alteração dinâmica de imagens de fundo
- Efeitos visuais em textos
- Criação de botões interativos
- Modo escuro e modo claro

##  Tecnologias utilizadas
- HTML5
- CSS
- JavaScript

##  O que aprendi
- Manipulação de CSS com JavaScript
- Manipulação do DOM
- Eventos de clique
- Alternância entre temas
- Estruturação de páginas web
- Organização de componentes visuais

##  Como executar
Basta abrir o arquivo html no navegador.
