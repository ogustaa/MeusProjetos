# Projeto Jogo do Número Secreto

Projeto simples desenvolvido para praticar lógica de programação com JavaScript com um mini jogo web de adivinhação.

O objetivo do jogo é descobrir o número secreto definido no código, caso você nao acerte o número, o site dara uma dica, ao fim do jogo, conta com quantas tentativas você acertou o número secreto.

> Observação: os arquivos HTML e CSS utilizados no projeto foram disponibilizados pela plataforma ALURA. O foco principal do projeto foi o desenvolvimento da lógica em JavaScript.

##  Tecnologias utilizadas
- JavaScript

##  O que aprendi
- Manipulação do DOM
- Criação de funções
- Estruturas condicionais
- Lógica de programação
- Interação com o usuário

##  Como executar
Basta abrir o arquivo .html no navegador.
