# Calculadora de Bhaskara

Projeto simples desenvolvido em Python para calcular raizes de uma equação de segundo grau utilizando a formula de Bhaskara.

##  Tecnologias utilizadas
- Python

##  O que aprendi
- Entrada de dados
- Estruturas condicionais
- Operações matemáticas
- Utilização da formula de Bhaskara
- Lógica de programação

##  Como executar
Execute o arquivo .py no VS CODE ou em outro lugar usando Python e informe os valores de A, B e C.
